### Controls
* W - Up
* S - Left
* A - Down
* D - Right
* Up - Camera Up
* Left - Camera Left
* Down - Camera Down
* Right - Camera Down
* Space - Jump/OK
* Left Shift - Run
* Escape - Pause
* F1 - Hide HUD
* F2 - Screenshot
* F3 - Open Console
