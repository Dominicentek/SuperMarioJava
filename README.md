### Controls
* W - Up
* S - Left
* A - Down
* D - Right
* Up - Camera Up
* Left - Camera Left
* Down - Camera Down
* Right - Camera Down
* Space - Jump/OK
* Left Shift - Run
* Escape - Pause
* F1 - Hide HUD
* F2 - Screenshot
* F3 - Open Console

### Console Commands
You can press Enter to execute a command<br>
or use the arrow keys to execute previous commands

* `coin &lt;set|add&gt; &lt;amount&gt;` - Manages the amount of coins
